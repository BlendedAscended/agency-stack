
# Agency Stack: Windmill + n8n with Caddy

Self-hosted automation stack with Windmill (workflow automation) and n8n (workflow automation) behind a Caddy reverse proxy.

## Quick Start

1. Clone repository
2. Copy `.env.example` to `.env` and configure
3. Run `docker compose up -d`
